# ♻️ Sistema de Gestão de Doações de Óleo de Cozinha Usado

## 📖 Sobre o Projeto

Este projeto foi desenvolvido em linguagem C com o objetivo de auxiliar no gerenciamento das doações de óleo de cozinha usado destinadas à usina de biocombustível da Uniara.

A proposta surgiu como uma aplicação prática dos conhecimentos técnicos e ferramentais adquiridos pelos alunos durante as disciplinas do semestre, permitindo o desenvolvimento de uma solução voltada para uma necessidade real da instituição.

---

## 🎯 Objetivo

Criar um sistema utilizando os conhecimentos técnicos e as ferramentas estudadas nas disciplinas do curso, fornecendo a gestão e o controle das doações de óleo de cozinha usado destinadas ao processamento na usina de biocombustível da Uniara.

Além de contribuir para o aprendizado dos alunos, o sistema busca apoiar iniciativas de sustentabilidade e reaproveitamento de resíduos, colaborando com a produção de biocombustível.

---

## ⚙️ Funcionalidades

* ✅ Cadastro de doadores;
* ✅ Registro das quantidades de óleo doadas;
* ✅ Armazenamento das informações em arquivo;
* ✅ Consulta dos registros cadastrados;
* ✅ Menu interativo para navegação do sistema;
* ✅ Persistência de dados entre execuções.

---

## 🛠️ Tecnologias Utilizadas

* Linguagem C
* Estruturas (`struct`)
* Funções
* Manipulação de arquivos
* Vetores e Strings
* Ambiente GCC / MinGW

---

## 🚀 Como Executar

### Compilar

```bash
gcc ColetaDeOleo.c -o ColetaDeOleo
```

### Executar

```bash
./ColetaDeOleo
```

No Windows:

```bash
ColetaDeOleo.exe
```

---

## 🌱 Impacto Ambiental

O descarte inadequado do óleo de cozinha pode causar sérios impactos ao meio ambiente, contaminando solos, rios e sistemas de esgoto.

Por meio do controle das doações, este projeto incentiva a destinação correta do óleo usado, contribuindo para a produção de biocombustível e para práticas mais sustentáveis.
